Connect your GitHub account

To publish updates to the site, you need a GitHub personal access token. This is only needed on your computer — you enter it once and it's saved.

Steps to create a token:

  1. Go to github.com/settings/tokens/new
  2. Give it a name like "MSC Vessel Watch"
  3. Set expiration to 1 year (or no expiration)
  4. Under Scopes, tick public_repo
  5. Click "Generate token" and copy it

Your token is stored only in this browser and never sent anywhere except GitHub's API.
