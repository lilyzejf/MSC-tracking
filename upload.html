Connect your accounts

You need two keys — one to save data to GitHub, one for AI-powered PDF reading. Both are stored only in your browser.

1. GitHub token — lets the tool publish updates to the site.

  1. Go to github.com/settings/tokens/new
  2. Give it a name like "MSC Vessel Watch"
  3. Set expiration to 1 year (or no expiration)
  4. Under Scopes, tick public_repo
  5. Click "Generate token" and copy it

2. Google Gemini API key — lets the tool intelligently read vessel names from any PDF format.

  1. Go to aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click "Create API key", copy it

Both keys are stored only in this browser and never committed to GitHub.
